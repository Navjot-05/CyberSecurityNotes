# Operations
# sort : rarranging the lines ascending or descending
        1. sorting based on the characters of the begining
            $ sort <filename>

        2. combines the two files and then sort the lines and display the output on the terminal
            $ cat file1 file2 | sort

        3. sort the lines in reverse order
            $ sort -r <filename>
        
        4. sorting based on the third field on each line instead of the begining
            $ sort -k 3 <filename>
