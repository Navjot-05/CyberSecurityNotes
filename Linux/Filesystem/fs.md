# Filesystems supported by Linux
    1. Conventional disk fs
        ext3, ext4, XFS, Btrfs, JFS, NTFS, vfat,etc


    2. Flash storage fs
        ubifs, jffs2, yaffs,etc

    3. Database fs

    4. Special purpose fs
        procfs, sysfs, tmpfs, squashfs, debugfs,etc

# FileSystem 
    /run/media : external removable media or disk partition are visible here once mounted
